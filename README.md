## Requirements
  * Mac
  * Xcode
  * [Cocoapods](https://cocoapods.org/)
    * `sudo gem install cocoapods`

## Setup
  1. `git clone` this repository
  2. `pod install`
  3. `open Publications.xcworkspace`
